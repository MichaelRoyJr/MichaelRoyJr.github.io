A document that contains a description of the C# programming language. Covers
the structure and syntax as well as features of the language. Discusses briefly
many applications C# is useful for.  Features a navigation pane rooted to the screen
to easily navigate sections (quite helpful as the document is long). Contains multiple
code examples. Uses a media query to shift navigation panel to the top of the screen
on mobile sized devices.

  -Navigation links fixed to the viewport
  -Embedded code samples
  -Media query to fit small screens better
