A fictional product landing page for the 2020 Chevrolet Corvette. This page contains
information about the product including photos, specifications, options and prices.
Also featured is an information gathering form with an offer to encourage the user
to submit basic info. This page would likely be used in response to a user click
from another page or advertisement.

  -Navigation bar links to sections of the page
  -Multiple levels of CSS Flexbox.
  -Email address verification
  -Embedded video
